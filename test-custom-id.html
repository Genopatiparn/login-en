ทดสอบสร้างข้อมูลพร้อม Custom ID






ตัวอย่างการใช้งาน API

1. สร้างข้อมูลพร้อม Custom ID:

curl -X POST http://localhost:3000/api/persons \
  -H "Content-Type: application/json" \
  -d '{
    "id": "EMP001",
    "firstName": "John",
    "lastName": "Doe",
    "email": "john@example.com"
  }'

2. ดึงข้อมูลด้วย Custom ID:

curl -X GET http://localhost:3000/api/persons/EMP001
curl -X GET http://localhost:3000/api/persons/88

3. ดึงข้อมูลด้วย MongoDB _id:

curl -X GET http://localhost:3000/api/persons/6943bd50f0c72b8a390def40

4. แก้ไขข้อมูลด้วย Custom ID:

curl -X PUT http://localhost:3000/api/persons/EMP001 \
  -H "Content-Type: application/json" \
  -d '{"firstName": "Jane"}'

5. ลบข้อมูลด้วย Custom ID:

curl -X DELETE http://localhost:3000/api/persons/EMP001

ข้อดีของ Custom ID:

  • ใช้เป็นรหัสพนักงาน เช่น EMP001, EMP002
  • ใช้เป็นรหัสนักเรียน เช่น STU2024001
  • ใช้เป็นรหัสสินค้า เช่น PROD001
  • ง่ายต่อการจำและใช้งาน
  • สามารถค้นหาด้วย Custom ID หรือ MongoDB _id ได้
